# Incubator-Controller
Adaptative PID with:
* 3x PRT sensor
   *  Precision of ±0.5°C
   *  Partnumber: adafruit 3984
   *  Easy to calibrate with water and ice at 0°C
*  driven by 3x MAX31865
   *  Resolution of 0.03125°C
   *  Partnumber: adafruit 3648
* 1x MHS3528 screen with touch
  * Use Bodmer TFT_eSPI library
  * Touch controller: XPT2046
  * Display controller: ILI9486
* 2x SHT30 temperature and humidity sensor
  * Partnumber: adafruit 5064
* 1x esP32-S3 with some relay for contolling everything
  * Partnumber: Waveshare esP32-S3-Relay-6CH
